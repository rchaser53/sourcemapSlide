数値の規則

-231 〜 231-1 が表現できる

							
/* 以下の文字列は64文字存在する。「A」を0, 「/」を63とする */
ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz
0123456789+/

任意の一単語は6桁の桁の2進数を意味する

A => 000000

C => 000010

Q => 010001

g => 100000

6桁の数字の意味

上1桁 continuous bit 次の単語のbitを5bitシフトする
中4桁 number bit 実際の値を示す
下1桁 sign bit 正の値か負の値か

ターゲット: G

2進数: 000110

結論: 3

continuous bit: 0

number bit: 0011

sign bit: 0
